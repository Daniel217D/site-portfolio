# Project Specification

## Статус

Draft v0.1

---

# Назначение

Статический сайт-портфолио для демонстрации ML-проектов.

Сайт содержит:

* главную страницу;
* каталог проектов;
* страницы отдельных проектов;
* презентации на базе Reveal.js;
* интерактивные демо для взаимодействия с ML-моделями через внешний API.

---

# Подход к разработке

Проект использует Spec-Driven Development.

Перед реализацией функциональности необходимо обновить соответствующую спецификацию.

Код является реализацией спецификаций, а не источником требований.

---

# Технологический стек

## Frontend

* Nuxt.js
* Vue 3
* TypeScript
* Reveal.js

## Качество кода

* ESLint
* Vitest
* Playwright

---

# Рендеринг

Тип проекта:

```text
Static Site Generation (SSG)
```

Сборка:

```bash
npm run generate
```

Результат:

```text
.output/public
```

Production не использует Node.js runtime.

---

# Разработка

Основная среда разработки:

```text
Docker
```

Дополнительно должна поддерживаться локальная разработка без Docker.

---

# README

В проекте должен присутствовать README.md.

README должен содержать:

* описание проекта;
* требования к окружению;
* локальный запуск;
* запуск через Docker;
* список команд;
* описание переменных окружения;
* сборку проекта;
* обзор структуры проекта.

---

# Agent Support

Проект ориентирован на разработку с использованием AI-агентов.

Необходимо создать набор переиспользуемых инструкций:

```text
.codex/
└── skills/
    ├── project-structure.md
    ├── fsd-rules.md
    ├── nuxt-page.md
    ├── demo-component.md
    ├── reveal-presentation.md
    ├── testing.md
    ├── ci-cd.md
    └── git-workflow.md
```

Назначение:

* фиксация архитектурных правил;
* стандартизация реализации;
* снижение количества ручных инструкций агентам.
