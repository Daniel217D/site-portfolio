# Модель проекта

На текущем этапе используется минимальная модель проекта.

```ts
interface Project {
  project_name: string
  title: string
}
```

Пример:

```ts
export const projects: Project[] = [
  {
    project_name: 'sentiment-analysis',
    title: 'Sentiment Analysis'
  }
]
```

---

# Правила описания проектов

* `project_name` используется в маршрутизации и для поиска проекта.
* `title` используется для отображения в каталоге и на странице проекта.
* Простое текстовое описание проекта размещается прямо в шаблоне страницы проекта.
* Остальное содержимое проекта на текущем этапе также допускается размещать непосредственно в коде страницы проекта.

---

# Расширение модели

Текущая модель является минимальной и может быть расширена позднее отдельной спецификацией без нарушения базовой схемы маршрутизации.
