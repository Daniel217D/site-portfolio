# Portfolio ML Showcase

## Статус

Draft v0.1

---

# Назначение

Этот документ является верхнеуровневой рамочной спецификацией проекта.

Статический сайт-портфолио для демонстрации ML-проектов.

Сайт содержит:

* главную страницу;
* каталог проектов;
* страницы отдельных проектов;
* презентации на базе Reveal.js;
* интерактивные демо для взаимодействия с ML-моделями через внешний API.

---

# Подход к разработке

Проект использует Spec-Driven Development.

Перед реализацией функциональности необходимо обновить соответствующую спецификацию.

Код является реализацией спецификаций, а не источником требований.

# Reveal.js

Для презентаций используется Reveal.js.

Презентации могут содержать:

* анимации;
* пошаговое раскрытие контента;
* интерактивные элементы;
* графики;
* визуализации.

Презентация является частью страницы проекта и встраивается как отдельный компонент.

Способ интеграции с Nuxt определяется в процессе реализации.

# Agent Support

Проект ориентирован на разработку с использованием AI-агентов.

Необходимо создать набор переиспользуемых инструкций:

```text
.codex/
└── skills/
    ├── project-structure.md
    ├── fsd-rules.md
    ├── nuxt-page.md
    ├── demo-component.md
    ├── reveal-presentation.md
    ├── testing.md
    ├── ci-cd.md
    └── git-workflow.md
```

Назначение:

* фиксация архитектурных правил;
* стандартизация реализации;
* снижение количества ручных инструкций агентам.
